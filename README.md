# AiRepoBlueprint

A copy-only AI instruction blueprint for new repositories.

To use it, copy these Markdown files and folders into a new repo:

- `AGENTS.md`
- `CLAUDE.md`
- `GEMINI.md`
- `CHANGELOG.md`
- `docs/ai/`
- `.claude/`
- `.agents/`

This repo has no scripts, no generator metadata, and no GitHub Actions.
