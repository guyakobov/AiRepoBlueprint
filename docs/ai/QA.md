# QA

Before publishing any AI-made change to `main`, the AI assistant must run the project in local dev mode.

The assistant should:

1. Start the local dev server.
2. Open the localhost URL.
3. Check that the requested change was fulfilled.
4. Check that the page or app still works.
5. Fix issues before pushing to `main`.

If the project has no frontend or localhost mode, use the closest available validation command.
